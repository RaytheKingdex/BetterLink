# Diagram Explanations

This folder contains written explanations for each PlantUML diagram in `docs/diagrams/`.

## Files

- `class-diagram.md`
- `erd.md`
- `sequence-diagram.md`
- `use-case-diagram.md`
- `state-machine-diagram.md`
- `system-architecture-diagram.md`
- `flowchart.md`
- `activity-diagram.md`

Each explanation describes the purpose of the diagram, the main elements shown, and how the diagram maps to the BetterLink codebase and project design.
